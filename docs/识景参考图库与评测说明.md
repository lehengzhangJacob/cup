# 识景参考图库与评测说明

## 素材来源与授权

管理员在“知识库管理 → 识景参考图库”中，为每个具体子景点上传参考截图，并记录来源链接、视角和天气说明。灵山胜境、拈花湾禅意小镇的 720 云 VR 可作为截图来源；仅上传团队确认有权用于比赛演示与模型复核的画面，不自动下载、抓取或二次分发第三方素材。

建议每个子景点至少 5 张：正面/侧面/远景、不同光照、关键建筑或雕塑特写。文件会统一校验、纠正方向并转为 JPEG。

## 识景链路

1. 游客图片质量检查与规范化；
2. GLM-4V 从景区目录产生最多 3 个候选；
3. 对候选及已确认定位点，取本地授权参考图做第二次多图视觉复核；
4. 若置信度不足，前端让游客确认，不能直接当作事实；
5. 对确认的景点由 RAG 给出带资料来源的讲解。

GPS/二维码只作为先验，不能单独把视觉结果升格为高置信度。

## 评测集

- 识景：每个子景点至少 10 张未入库测试图，记录 Top-1、Top-3、拒识准确率和平均/P95 时延；
- RAG：使用 `llm/scripts/evaluate_fact_benchmark.py` 生成可审计报告。15 题文件仅为冒烟基线，赛题提交前补充不少于 80 道专家题；
- 情绪：分别报告文本、音频+文本的 Macro-F1、各类召回、混淆矩阵与 P95 时延。游客端未采集人脸视频时，材料中必须表述为“音频—文本多模态情绪分析”。
