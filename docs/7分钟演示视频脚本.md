# 7 分钟产品演示视频脚本

| 时间 | 画面 | 解说重点 |
|---|---|---|
| 0:00–0:30 | 封面、景区游客画面 | 导游供需、单向讲解和管理盲区，提出“灵山小向导”。 |
| 0:30–1:10 | 架构动效或 PPT | 游客端、AI 网关、本地知识库、多模态模型、数字人和管理后台。 |
| 1:10–2:10 | HTTPS 游客端，现场语音提问 | 问“灵山大佛有多高、用了多少铜”，展示语音识别、首句快速开口和口型同步。 |
| 2:10–2:50 | 连续追问并打断 | 展示流式回答、情绪变化、引用资料和新问题打断旧播报。 |
| 2:50–3:35 | 兴趣选择与路线推荐 | 分别切换亲子/历史兴趣，展示路线、时长、站点和讲解重点。 |
| 3:35–4:15 | 拍照识景、当前位置按钮 | 展示 GLM-4V 识别后经本地知识库校准，以及弱定位讲解。 |
| 4:15–4:40 | 提交 5 分或意见 | 展示游客满意度和文字建议进入数据闭环。 |
| 4:40–5:25 | 管理后台数据概览/报告 | 展示今日人次、热门问题、情感趋势、响应延迟和系统建议。 |
| 5:25–5:55 | 上传一份 TXT/DOCX | 展示自动重建知识库；随后用游客端询问新知识验证实时生效。 |
| 5:55–6:20 | 数字人管理 | 修改名称、声音或已安装形象，说明新会话生效和文化定制能力。 |
| 6:20–6:45 | 测试结果页 | 标准题 15/15、事实准确率 100%；端到端首次开口 3373ms。 |
| 6:45–7:00 | 总结页 | “让景区每位游客有专属导游，让每次互动成为运营依据。” |

录制前检查 GPU、8001/8443/8010、麦克风权限和证书；准备文字输入作为语音环境嘈杂时的备份。
